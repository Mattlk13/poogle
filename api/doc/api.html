Poogle API documentation version v1

Poogle is a meta search engine that gather searches from different porn sites and display them into one only convivial interface. It allows you to find the porn you're looking for faster and gives you the direct link to the original videos. It is for now intended to work with a limited number of non-free big porn websites.

post /user

Add a new user

  • Request
  • Response

Query Parameters

  • email: required (string)
  • userName: required (string)
  • receivesNews: (boolean - default: false)

    Whether or not the user receives news about the service (updates, improvements, ...)

  • services: (array)

    Array of services id.

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
  userName: "jeanmi",
  email: "jeanmi.const@example.com",
  receivesNews: false,
  services: [1, 2]
}
